# PowerEye Frontend

**Home Energy Monitoring System (HEMS)** is a system that collects energy data from various household appliances, processes and analyzes them, and presents the findings to users through an intuitive mobile interface to give them insights about their energy consumption for better energy sustainability.

## Application Features (Frontend):
This repository contains the **Mobile Application** built with React Native, focusing on delivering a smooth and interactive user experience:
* **Interactive Data Visualization:** Dynamic charts and graphs to monitor energy consumption.
* **Appliance & Room Management:** User-friendly interfaces to add, track, and organize smart plugs within different home sectors and rooms.
* **Smart Dashboard:** A clean UI that displays energy goals, real-time consumption, cost predictions, and AI-driven recommendations.
* **Alerts & Notifications:** Visual feedback and push notifications to keep users informed about their consumption limits and goals.
* **Backend Repository:** [[https://github.com/rem2718/PowerEye-backend.git]](https://github.com/rem2718/PowerEye-backend.git)

## How to Run It:
To run the mobile application on your device or emulator:
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
3. Start the project using Expo:
   ```bash
   npx expo start -c
4. Scan the QR code with the Expo Go app (available on iOS and Android) to view the app.
